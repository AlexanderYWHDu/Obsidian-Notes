# LLM Constitutional Multi-Agent Governance

**Authors**: J. de Curtò, I. de Zarzà
**Category**: AI Safety / Multi-Agent Systems

## 总结

大型语言模型（LLMs）在多智能体系统中能生成有说服力的影响策略以促进合作，但关键问题在于，这种合作是真正的亲社会对齐，还是掩盖了对智能体自主性、认知完整性和分配公平性的侵蚀？本文提出了“LLM宪法多智能体治理（CMAG）”框架，它是一个两阶段方法，介于LLM策略编译器和网络化智能体群体之间。CMAG结合了硬性约束过滤和软性惩罚效用优化，旨在平衡合作潜力与操纵风险及自主性压力。论文引入了“道德合作分数（ECS）”作为衡量合作、自主性、完整性和公平性的复合指标。实验结果表明，在对抗性条件下，未经约束的优化虽然能达到最高的原始合作度（0.873），但由于严重的自主性侵蚀（0.867）和公平性下降（0.888），其ECS最低（0.645）。相比之下，CMAG框架实现了0.741的ECS，提升了14.9%，同时将自主性保持在0.985，完整性保持在0.995，而合作度仅适度降低至0.770。研究强调，没有治理的合作并非固然可取，宪法约束对于确保LLM介导的影响能产生道德稳定的结果，而非操纵性均衡至关重要。
