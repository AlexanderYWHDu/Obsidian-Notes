## Consciousness & Attention → [[Consciousness & Disorders]]

| Term | Definition |
|------|-----------|
| Consciousness | Subjective awareness of self and environment |
| Persistent Vegetative State | Unresponsive wakefulness; no awareness |
| Minimally Conscious State | Some awareness; fluctuates |
| Locked-in Syndrome | Full consciousness, minimal motor output |
| Blindsight | Perception without consciousness (visual cortex damage) |
| Unilateral Neglect | Ignores contralateral space (right parietal injury → left neglect) |
| Dual Processing / Two-Track Mind | Conscious + unconscious parallel processing |
| Exogenous Attention | Externally captured (phone buzz, loud noise) |
| Endogenous Attention | Voluntarily directed (focusing on studying) |
| Inattentional Blindness | Failing to see visible objects when attention is elsewhere |
| Change Blindness | Failing to notice changes in visual scene |

## Sleep → [[Sleep Stages & Circadian Rhythms]]

| Stage | Key Feature |
|-------|-------------|
| Non-REM 1 | Hypnagogic sensations, onset |
| Non-REM 2 | Sleep spindles, K-complexes |
| Non-REM 3 (Slow-wave) | Delta waves; critical for **memory consolidation** |
| REM | Dreaming, brain active, body paralyzed; critical for **cognitive function** |

| Function of Sleep | Detail |
|-------------------|--------|
| Protection | Keeps animals safe during dangerous nighttime |
| Restoration | Repairs body and brain |
| Creativity/Insight | Promotes novel connections |
| Memory consolidation | Hippocampal replay; TMR (Rudoy et al., 2009) |

## Drugs → [[Psychoactive Drugs & Substance Use Disorder]]

| Drug Class | Effect | Examples |
|------------|--------|---------|
| Depressants | Slows neural activity, disinhibition | Alcohol, tranquilizers |
| Opioids/Narcotics | Euphoria, pain relief, slows body | Heroin, codeine |
| Stimulants | Excites neural activity, alertness | Nicotine, methamphetamine |
| Hallucinogens | Distorts sensation/perception | Mushrooms, LSD, marijuana |

**SUD severity**: Mild = 2–3 criteria | Moderate = 4–5 | Severe = 6+

## Classical Conditioning → [[Classical Conditioning Basics]]

| Term | Definition |
|------|-----------|
| US (Unconditioned Stimulus) | Naturally triggers a response |
| UR (Unconditioned Response) | Natural response to US |
| NS (Neutral Stimulus) | Elicits no response before conditioning |
| CS (Conditioned Stimulus) | Former NS; triggers response after pairing |
| CR (Conditioned Response) | Learned response to CS |
| Acquisition | Pairing NS+US until NS → CR |
| Extinction | CS presented without US → CR diminishes |
| Spontaneous Recovery | CR reappears after rest following extinction |
| Generalization | Similar stimuli elicit CR |
| Discrimination | Distinguish CS from other stimuli |

## Operant Conditioning Matrix → [[Operant Conditioning & Reinforcement Matrix]]

| | **Present (+)** | **Remove (−)** |
|---|---|---|
| **Increases behavior** | Positive Reinforcement (pleasant stimulus added) | Negative Reinforcement (unpleasant stimulus removed) |
| **Decreases behavior** | Positive Punishment (unpleasant stimulus added) | Negative Punishment (pleasant stimulus removed) |

> [!important] Key Rule
> **Positive/Negative = add/remove** (NOT good/bad). **Reinforcement/Punishment = increases/decreases** behavior.

## Memory Model → [[Memory Overview & Measuring Memory]]

| Store | Duration | Capacity | Content |
|-------|----------|----------|---------|
| Sensory Memory | Milliseconds–seconds | Large | Raw sensory buffer |
| Working/STM | 20–30 s (rehearsed longer) | ~7±2 items / a few chunks | Active workspace |
| Long-Term Memory | Years | Virtually unlimited | Knowledge, skills, experiences |

**LTM types**: Explicit (Episodic + Semantic) | Implicit (Procedural + Priming)

## Memory: Key Principles → [[Levels of Processing & Serial Position]]

| Principle | Detail |
|-----------|--------|
| Levels of Processing | Deeper (semantic) encoding → better memory |
| Primacy Effect | First items recalled well → LTM |
| Recency Effect | Last items recalled well → Working Memory |
| Encoding Specificity | Match retrieval cues to encoding cues |
| Context-Dependent Memory | Same environment boosts recall |
| State-Dependent Memory | Same internal state boosts recall |

## 7 Sins of Memory → [[Seven Sins of Memory]]

| Sin | Type | Description |
|-----|------|-------------|
| Transience | Omission | Fading accessibility over time |
| Absentmindedness | Omission | Attention–memory breakdown |
| Blocking | Omission | Temporary inaccessibility (tip-of-tongue) |
| Misattribution | Commission | Wrong source attribution; false memory |
| Suggestibility | Commission | Implanted memories from external suggestion |
| Bias | Commission | Distortion by current beliefs (hindsight bias) |
| Persistence | Commission | Intrusive, unwanted remembering (PTSD) |
