# Emotional Memory & Retrieval Principles (★★)

#memory #memory-retrieval

## Overview Table

| Concept | Key Point |
|---------|-----------|
| Encoding Specificity | Better recall when retrieval cues match encoding cues |
| Context-Dependent Memory | Same physical environment boosts recall |
| State-Dependent Memory | Same internal state (mood/drug) boosts recall |
| Mood-Congruent Memory | Easier to recall memories matching current mood |
| Flashbulb Memory | Vivid emotional memory; feels accurate but still distorts |

---

## Encoding Specificity Principle (Tulving)

> [!important] Core Principle
> **The more closely retrieval cues match encoding cues, the better the information will be recalled.**

```
ENCODING CONTEXT ──matches──► RETRIEVAL CONTEXT
                      │
                      ▼
               BETTER RECALL
```

---

## Context-Dependent Memory

- **External environment** at encoding can serve as a retrieval cue
- Classic study: Divers learned word lists underwater or on land; recalled better in the same location
- Application: Study in the same room where you'll take the test (if possible)

---

## State-Dependent Memory

- **Internal state** (mood, arousal, drug state) at encoding serves as a retrieval cue
- Examples:
  - If you learned something while drinking coffee, recalling in a caffeinated state might help
  - People in a sad mood recall sad memories more easily

### Mood-Congruent Memory
- You more easily recall memories that match your **current emotional state**
- Example: When depressed, you more easily recall sad or negative memories — reinforces the depressive state

---

## Flashbulb Memories

- **Definition**: Vivid, detailed memories of emotionally significant or surprising moments
- Examples: Where you were during 9/11, a major personal event
- Characteristics:
  - Feel highly accurate and permanent
  - Created by amygdala-boosted encoding
- **Reality**: Still subject to forgetting and distortion, just like any other memory

> [!warning] Trap
> Flashbulb memories **feel** accurate but are NOT necessarily more accurate than ordinary memories — they can be wrong in specific details.

---

## Memory as Reconstruction

- Human memory is **NOT like a video recorder**
- We **reconstruct** the past: more like an artist's rendering than a faithful record
- "Forgetting is adaptive" — selective forgetting allows efficient processing

---

## Exam/Test Patterns

| Scenario/Keyword | Answer |
|-----------------|--------|
| Studying on campus → doing better on campus test | **Context-dependent memory** |
| Recalls sad events when sad | **Mood-congruent memory / state-dependent** |
| "Where I was when X happened" vivid but possibly wrong | **Flashbulb memory** |
| Better recall when retrieval cues match encoding | **Encoding specificity principle** |
| Memory is like a video playback | **FALSE** — memory is reconstructive |
