# Long-Term Memory Types (★★★)

#memory #memory-storage

## Overview Table

| Type | Subtypes | Description | Conscious? |
|------|----------|-------------|-----------|
| **Explicit (Declarative)** | Episodic, Semantic | Facts and events you can articulate | Yes |
| **Implicit (Nondeclarative)** | Procedural, Priming | Skills and associations; hard to articulate | No |

---

## Long-Term Memory Types

```
LONG-TERM MEMORY
├── EXPLICIT (Declarative) ─ can be consciously stated
│   ├── Episodic: Personal experiences ("I went to Paris")
│   └── Semantic: General knowledge ("Paris is the capital of France")
│
└── IMPLICIT (Nondeclarative) ─ hard to consciously articulate
    ├── Procedural: Motor skills (riding a bike)
    └── Priming: Activation of associations in memory (unconscious)
```

---

## Explicit Memory

### Episodic Memory
- **Personal experiences** with time, place, and emotional context
- "When" and "where" memories
- Example: your first day of university

### Semantic Memory
- **General knowledge** about the world, facts, concepts
- Not tied to personal experience
- Example: knowing that psychology is a social science

---

## Implicit Memory

### Procedural Memory
- **Motor skills** and habitual behaviors
- Learned through practice; hard to verbally describe
- Examples: typing, cycling, playing an instrument

### Priming
- **Activation of associations** in memory — typically unconscious and implicit
- Prior exposure to one stimulus influences response to a related stimulus
- Example: seeing "bread" makes you faster to recognize "butter"

---

## Emotional Memory & the Amygdala
- The **amygdala** boosts memory processing during **emotional or stressful** times
- This is why emotional events are often remembered more vividly
- **Flashbulb memories**: vivid memories of emotionally significant moments (e.g., 9/11, personal crises)
  - We *think* they are highly accurate, but they are still subject to forgetting and distortion like any other memory

## Case Study: Patient HM

> [!important] Why HM Matters
> Bilateral removal of hippocampus → severe **anterograde amnesia** → proved different memory systems exist

| Memory Test | HM's Performance |
|-------------|-----------------|
| Working memory tasks | Almost **normal** |
| Remembering he had done the task | **Cannot** (no new LTM) |
| Procedural learning (motor tasks) | **Intact** — could learn new skills |
| Remembering learning the skill | **Cannot** |

- **Anterograde amnesia**: Cannot form NEW long-term memories
- **Retrograde amnesia** (comparison): Cannot recall OLD memories before injury
- HM could learn new motor skills (implicit/procedural) but couldn't remember doing them (explicit)

## Exam/Test Patterns

| Scenario/Keyword | Answer |
|-----------------|--------|
| "I remember my first bike ride" | **Episodic memory** |
| Knowing Paris is the capital of France | **Semantic memory** |
| Riding a bike without thinking about it | **Procedural memory** |
| Seeing "bread" → faster recognition of "butter" | **Priming** |
| HM's deficit: couldn't form new LTM | **Anterograde amnesia** |
| HM's intact ability: new motor skills | **Procedural/implicit memory** |
| Flashbulb memories: are they accurate? | Not necessarily — still subject to distortion |
