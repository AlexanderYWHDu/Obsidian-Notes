# Inattentional & Change Blindness (★★)

#consciousness #attention

## Overview Table

| Concept | Key Point |
|---------|-----------|
| Inattentional Blindness | Fail to see visible objects when attention is elsewhere |
| Change Blindness | Fail to notice changes in a scene |
| Relationship | Change blindness is one type of inattentional blindness |

---

## Inattentional Blindness

- **Definition**: Failing to see clearly visible objects when attention is directed elsewhere
- Classic demonstration: **Gorilla in basketball video** (Simons & Chabris) — ~50% of observers miss a person in a gorilla suit walking through the scene

### Real-World Study
- Police officers jogging: a fight was visible for ~30 seconds starting ~20 seconds into their run
- Only **56% noticed the fight** — even trained law enforcement missed it when their attention was focused elsewhere
- Demonstrates that attention is a limited, selective resource

---

## Change Blindness

- **Definition**: Failure to notice that something in the visual scene has changed
- A *sub-type* of inattentional blindness
- Example: In film, different actors in the same shot between cuts; many viewers don't notice

```
Frame A ──── cut ──── Frame B
  [Person has different coat]
        ↓
Most viewers: DO NOT NOTICE
```

---

## Why These Occur (Adaptive Perspective)

> [!tip] Adaptive Value
> These aren't failures — they're features. The brain cannot process everything. By filtering, it allows efficient conscious processing of what matters most.
> The **two-track mind** handles this: unconscious processing handles the rest, only flagging things for consciousness when relevant.
