# Working Memory & Chunking (★★★)

#memory #memory-encoding

## Overview Table

| Concept | Key Point |
|---------|-----------|
| Short-Term Memory (STM) | Simple storage buffer; ~7±2 items |
| Working Memory (WM) | Active workspace: storage + manipulation |
| Chunk | Meaningful unit of information (size varies) |
| Method of Loci | Mnemonic: place items along a familiar mental path |
| PAO System | Memory competitor technique: Person-Action-Object |

---

## Short-Term Memory → Working Memory

### Classic STM (Miller, 1956)
- **"Magical number 7 ± 2"**: capacity of roughly 5–9 items
- Simple storage and retrieval buffer

### Working Memory (Updated Model)
- More than just storage — includes **active processing**
- **Working memory**: conscious, active processing of both:
  1. Incoming sensory information
  2. Information retrieved from long-term memory

| | STM | Working Memory |
|---|---|---|
| Task | Simple storage | Storage + manipulation |
| Example | Digit span | Equation task while remembering numbers |

---

## Duration of STM/WM
- Without rehearsal: info decays in **~20–30 seconds**
- Rehearsal (repeating the info) keeps it active

---

## Capacity & Chunks

- **Chunk** = a meaningful unit; can be small (single letter) or large (a whole phrase)
- The real limit is **number of chunks**, not individual units
- Example: "FBI CIA IRS" = 9 letters but only **3 chunks** if you know the acronyms

```
"monkey child wildly zoo jumped city ringtail young"
= 8 words if treating each as a chunk
= fewer chunks if grouped meaningfully
```

### Chunking in Chess (Chase & Simon, 1973)
- Chess masters remember board positions much better than beginners — they see **meaningful chunks** (configurations), not individual pieces

---

## Mnemonics: Chunking Strategies

### Method of Loci (Memory Palace)
- Mentally **place** objects along a **familiar path** (e.g., your home, your body)
- At recall: mentally "walk" the path and "see" each object
- Used by ancient orators; still used by modern memory champions

### PAO System (Person–Action–Object)
- Every two-digit number (00–99) is encoded as a Person doing an Action on an Object
- Example: 14 = Taylor Swift singing into a microphone; 15 = Messi kicking a soccer ball
- Six digits of pi = one vivid image placed in the first "room" of a memory palace

> [!tip] Memory Competition Record
> 12.74 seconds to memorize a shuffled deck of cards using chunking + memory palace
