## What is Sensory Memory?

- **Definition**: The immediate, very brief recording of sensory information in the memory system
- Acts as a **buffer** that holds sensory info just slightly longer than the stimulus lasts
- Allows us to perceive continuity (e.g., movies appear as moving rather than still frames)
- Content: raw, unprocessed sensory data

---

## Sperling's Experiment (1960)

### Whole Report Method
- Flash a grid of letters (e.g., 3 rows × 4 columns) for 50 ms
- Ask participant to recall ALL letters
- **Result**: ~4.5 items recalled on average
- Conclusion: Capacity = ~4.5 items? But participants reported seeing more!

### Partial Report Method
- Flash grid → immediately play a tone (high/mid/low) → participant reports only THAT row
- **Results**:
  - 100% correct for a 3×3 array
  - 75% correct for a 3×4 array

### Conclusion
- **Both earlier estimates were partially right**:
  - All letters ARE briefly available (capacity > 4.5)
  - BUT they decay extremely fast — by the time you report all, earlier ones have already faded

```
SPERLING'S INSIGHT:
Flash of 12 letters
    │
    ▼ (all 12 available for ~fraction of a second)
Partial report: ~75-100% correct
    ↓ (milliseconds later, decay)
Whole report: only ~4.5 recalled
```
