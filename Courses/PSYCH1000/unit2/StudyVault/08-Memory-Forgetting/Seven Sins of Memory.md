# Seven Sins of Memory (★★★)

#memory #memory-forgetting #memory-construction

## Overview Table

| Sin | Type | One-line Summary |
|-----|------|-----------------|
| Transience | Omission | Fading over time |
| Absentmindedness | Omission | Attention–memory failure |
| Blocking | Omission | Temporary retrieval failure |
| Misattribution | Commission | Wrong source for a memory |
| Suggestibility | Commission | Outside influence creates false memory |
| Bias | Commission | Current beliefs distort past memories |
| Persistence | Commission | Intrusive, unwanted remembering |

> [!important] Key Insight
> All 7 sins arise from **normally adaptive** memory functioning — they are side effects of a useful system, not pure failures.

---

## 3 Sins of Omission (Forgetting)

### 1. Transience
- **Decreasing accessibility of information over time**
- Could be decay or interference
- Ebbinghaus's **forgetting curve**: rapid initial loss, then leveling off

#### Interference (2 types)
```
PROACTIVE (forward-acting):
OLD memory interferes with NEW memory
Example: Writing "2025" as the date in 2026
         More prior word lists → lower recall for new list

RETROACTIVE (backward-acting):
NEW memory interferes with OLD memory
Example: Learned new address → can't recall old address
```

### 2. Absentmindedness
- **Breakdown at the interface of attention and memory**
- Info not encoded because attention was elsewhere
- Examples: inattentional blindness, change blindness
- Yo-Yo Ma forgot his $2.5 million cello in a NYC cab

### 3. Blocking
- **Temporary inaccessibility** of information that IS stored
- More info is available than accessible at any moment
- **Tip-of-tongue (TOT) phenomenon**: know you know it, can't retrieve it
  - Feel close to the answer; partial information available; eventually may pop up

---

## 3 Sins of Commission (Distortion)

### 4. Misattribution
- Attributing a memory or idea to the **wrong source**
- **Source memory confusion**: confusing WHO told you something
- **False memory**: remembering something that didn't happen
- Roediger & McDermott (1995): Presented word lists associated with "sleep" → **"sleep" falsely recognized 72%** of the time
- Cryptomnesia: thinking you originated an idea that you actually heard before

### 5. Suggestibility
- **Implanted memories** resulting from suggestion or misinformation
- **Loftus & Palmer (1974)**: Car accident video + question with "smashed" vs. "hit"
  - "Smashed" → 32% falsely recalled seeing broken glass
  - "Hit" → only 14% false recall
- Suggestive procedures (visualization, social influence) create false memories in **20–30%** of participants
- Even vivid autobiographical false memories can be implanted (Jean Piaget example)

### 6. Bias
- **Retrospective distortions** produced by current knowledge, beliefs, or feelings
- **Hindsight bias**: "I knew it all along" — past judgment revised in light of outcome
- Top-down influences: memory is filtered through current worldview
- People have belief-consistent false memories (Murphy et al., 2019 — Ireland referendum)

---

## 1 Sin of Remembering

### 7. Persistence
- **Intrusive, unwanted** remembering of events you'd rather forget
- Example: **PTSD** — unrelenting, intrusive memories that feel as raw as the original event
- "Like having open wounds that are just a part of you"
- Adaptive origin: persistence ensures emotionally significant events remain memorable

---

## Recent Addition: Verisimilitude

### (8) Verisimilitude (Yilmaz et al., 2025)
- The appearance of **fluent, verbatim episodic memories being true** — even when they aren't
- Relevant to **eyewitness testimony**: recollections at trial carry enormous weight
- As of 2026: **255 overturned convictions** for wrongful convictions
  - 62% involved **eyewitness misidentification**
  - 29% involved **false confessions** (including coercive tactics)

## Exam/Test Patterns

| Scenario/Keyword | Answer |
|-----------------|--------|
| Writing last year's date | **Proactive interference** (old interferes with new) |
| Can't recall old address after moving | **Retroactive interference** (new interferes with old) |
| "Sleep" falsely recognized in Roediger study | **Misattribution / false memory** |
| "Smashed" → more false glass memories | **Suggestibility (Loftus & Palmer)** |
| Knowing the word but can't say it | **Blocking / tip-of-tongue** |
| "I knew it all along" | **Hindsight bias** (Bias sin) |
| Uncontrollable PTSD flashbacks | **Persistence** |
| Classify: 3 sins of omission | Transience, Absentmindedness, Blocking |
| Classify: 3 sins of commission (distortion) | Misattribution, Suggestibility, Bias |
