# False Memory & Suggestibility (★★★)

#memory #memory-construction

## Overview Table

| Concept | Key Point |
|---------|-----------|
| False Memory | Remembering something that didn't happen |
| Misinformation Effect | Post-event info alters memory of the original event |
| Source Monitoring Error | Attributing memory to wrong source |
| Eyewitness Testimony | Highly unreliable; subject to all memory distortions |

---

## Loftus & Palmer (1974) — Classic Study

### Method
- Participants watched a film of a car accident
- Different groups received different questions:
  - Group A: "How fast were the cars going when they **smashed** into each other?"
  - Group B: "How fast were the cars going when they **hit** each other?"

### Results
- **"Smashed"** group: estimated higher speed + **32%** falsely recalled seeing broken glass
- **"Hit"** group: **14%** false recall of broken glass
- (There was NO broken glass in the film)

> [!important] Conclusion
> The **wording of a question** can implant false memories. Post-event misinformation alters the original memory trace.

## How False Memories Form

```
Original Event
    │
    ▼
Encoding (incomplete, inferential)
    │
    ▼
Post-event information (leading questions, suggestions,
                         social influence, visualization)
    │
    ▼
Reconstructed Memory ← contains both real + suggested elements
```

- **Source monitoring confusion**: the brain doesn't always correctly tag WHERE information came from
- Something you heard, imagined, or were told can be misremembered as something you *experienced*

## Roediger & McDermott (1995) — DRM Paradigm

- Participants heard word lists (e.g., bed, rest, awake, tired, dream...)
- All associated with a **critical word** ("sleep") that was NEVER in the list
- **Result**: "sleep" falsely recognized **up to 72%** of the time
- Shows how strong associative networks create false memories spontaneously

## Jean Piaget Example
- Piaget had a vivid memory of an attempted kidnapping as a toddler
- Believed this memory for 15 years
- Later discovered: his nurse had **invented the story** and he had heard it as a child
- Memory = heard story + constructed "visual" scene → false episodic memory

## Eyewitness Memory & Legal Implications

> [!danger] Eyewitness Testimony Is Unreliable
> Despite feeling confident, eyewitness memories are subject to all 7 memory sins.

| Problem | Memory Sin Involved |
|---------|-------------------|
| Misidentifying a suspect | Misattribution |
| Influenced by leading questions | Suggestibility |
| Confident but wrong "flashbulb" memory | Verisimilitude / Bias |
| False confession (coercive tactics) | Suggestibility |

**As of 2026**: 255 wrongful conviction cases overturned in North America
- **62%** involved eyewitness misidentification
- **29%** involved false confessions

## Adaptive Side of False Memory

> [!tip] Why the system makes these errors
> - Source monitoring issues: **content is usually more important than source** → adaptive!
> - Construction allows us to fill in gaps, make abstractions, form gist-level representations
> - Forgetting allows quick judgments based on lingering feelings

| Scenario/Keyword | Answer |
|-----------------|--------|

| Thinking you came up with an idea someone else gave you | **Cryptomnesia / Misattribution** |
| "Sleep" falsely recognized in word list study | **DRM false memory / Misattribution** |
| Leading question in police interview | **Suggestibility risk** |
| Why false memories feel real | **Reconstruction + source monitoring failure** |
