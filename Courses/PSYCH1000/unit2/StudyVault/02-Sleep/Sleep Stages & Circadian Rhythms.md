# Sleep Stages & Circadian Rhythms (★★★)

#sleep

## Overview Table

| Stage | Brain Waves | Key Function |
|-------|-------------|-------------|
| Awake/Alert | Beta (fast) | — |
| Awake/Drowsy | Alpha | Relaxed wakefulness |
| Non-REM 1 | Theta | Onset; hypnagogic hallucinations |
| Non-REM 2 | Sleep spindles, K-complexes | Light sleep |
| Non-REM 3 (Slow-wave) | **Delta** (slow) | **Memory consolidation**, physical restoration |
| REM | Mixed (like waking) | **Dreaming**, cognitive function; body paralyzed |

---

## Circadian Rhythms

- **Definition**: Physical, mental, and behavioral changes following a natural **24-hour cycle** (not just sleep/wake)
- Regulated by: light exposure, body temperature, melatonin
- **Jet lag**: disruption of circadian timing from rapid time zone change
- **Social jet lag**: weekly mismatch between biological clock and social schedule (e.g., sleeping late on weekends)

```
CIRCADIAN CYCLE (24h)
 Midnight        Noon          Midnight
    │─────────────────────────────│
    ▓▓▓▓▓▓▓▓▓  SLEEP  ▓▓▓▓▓▓▓▓▓      AWAKE
         ↑ Core body temp drops; melatonin rises
```

> [!tip] Circadian Disadvantage in Sports
> Going **west** is usually MORE detrimental to sports performance (your body clock is ahead of local time; you're playing when it feels like nighttime).

## Sleep Architecture: 90-Minute Cycles

```
Sleep depth across the night (4–5 cycles):

Awake  ─────────────────────────────────────────────────
       ↘
N-REM 1 ─   ─   ─   ─   ─   ─   ─   ─   ─
         ↘ ↗ ↘ ↗
N-REM 2   ─────   ────   ────   ───
            ↘       ↘       ↘
N-REM 3      ████     ██      █        (more early)
REM                ▓▓▓    ▓▓▓▓▓  ▓▓▓▓▓▓  (more late)
```

- **Slow-wave sleep (N-REM 3)**: more in **early** part of night → memory consolidation
- **REM sleep**: more in **later** part of night → cognitive function, dreaming

---

## REM Sleep — Special Properties

- Heart rate rises; breathing rapid and irregular
- Brain activity closest to **waking state** (that's why dreams feel real)
- **Difficult to wake** from REM
- ~80% report dreaming during REM
- **Body is paralyzed** (motor cortex suppressed) — adaptive: prevents acting out dreams
- May linger into waking → **sleep paralysis**

> [!warning] N-REM 3 vs. REM — Don't Confuse!
> - N-REM 3 = physical restoration + **declarative memory** consolidation → **early** night
> - REM = cognitive function + **dreaming** → **late** night

---

## Sleep Across the Lifespan
- Newborns: ~16 hours/day; large proportion of REM
- Teenagers: circadian shift → biologically delayed sleep phase ("night owls")
- Adults: decreased slow-wave sleep with age
- Elderly: more fragmented sleep

## Exam/Test Patterns

| Scenario/Keyword | Answer |
|-----------------|--------|
| Memory consolidation happens in which stage? | **Non-REM 3 (slow-wave)** |
| Dreaming + brain active + body paralyzed | **REM** |
| Which is more prominent early in the night? | **Slow-wave (N-REM 3)** |
| Going west vs. east: which is worse for athletes? | **Going west** |
| Body clock follows a ___ cycle | **24-hour (circadian)** |
