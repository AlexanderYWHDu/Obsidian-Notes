## Why Do We Sleep?

1. **Protection**: Keeps us safe during the most dangerous time of day (evolutionary argument)
2. **Restore & repair**: Body and brain repair during sleep; growth hormone release
3. **Creative thinking/insight**: Sleep promotes novel connections between ideas
4. **Memory consolidation**: Active processing of daytime experiences

---

## Sleep & Memory — Two Views

### Initial View (Passive Protection)
- Sleep reduces interference: less new information coming in → less to disrupt encoding
- A *passive* benefit

### Current View (Active Consolidation)
- Sleep **actively** promotes memory reconsolidation
- "Memories may not lie dormant during sleep, but rather may be receiving regular exercise..." (Paller & Voss, 2004)

```
MEMORY CONSOLIDATION DURING SLEEP

Daytime learning
      │
      ▼
Hippocampus stores "event file"
      │
  SLEEP ──► Hippocampal REPLAY (rats: neural firing reactivates learning sequences)
      │
      ▼
Cortex forms durable long-term storage
(hippocampus no longer needed for retrieval)
```

---

## Key Evidence

### Hippocampal Replay in Rats
- During sleep, rats' hippocampal neurons fired in the same sequence as when the rat ran a maze — replaying the experience

### Targeted Memory Reactivation (TMR)
- **Rudoy et al. (2009)**: Humans learned object-location pairs, each with a sound cue
- During slow-wave sleep, some sounds were quietly replayed
- **Result**: Better recall for items whose sounds were replayed during sleep
- Conclusion: You can *selectively* strengthen specific memories during sleep

> [!tip] Consolidation Defined
> **Consolidation** = the process of turning encoded information into durable long-term memories through repeated reactivation, forming new associative connections in the cortex.

## Exam/Test Patterns

| Scenario/Keyword | Answer |
|-----------------|--------|
| Sleep helps memory because less interference | **Initial (passive) view** — now considered incomplete |
| Sleep helps memory by replaying experiences | **Active consolidation / hippocampal replay** |
| Playing sounds during sleep to boost recall | **Targeted Memory Reactivation (TMR)** |
| Where is a memory initially stored? | **Hippocampus** → transferred to **cortex** during sleep |
| After consolidation, is hippocampus needed? | **No** — cortex holds the long-term memory |
